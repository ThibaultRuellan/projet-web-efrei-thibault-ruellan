Maquette du site

Site vitrine du département informatique de l’EFREI

Cette maquette présente l’organisation générale du site, les pages prévues, la structure visuelle des écrans et les choix graphiques utilisés pour le projet.

Projet

Programmation Web TI402

Technologies

HTML5, CSS3, JavaScript

1. Plan du site

Le site est organisé autour de cinq pages principales afin de rendre la navigation simple et claire.

Formations
Vie étudiante
Contact
À propos

La version ordinateur utilise une mise en page large avec une navigation horizontale, tandis que la version mobile passe en colonne avec un menu responsive.

4. Structure des pages

Header

Le header contient le titre du site et la navigation principale vers toutes les pages.

Contenu principal

Les pages utilisent des sections, des cartes, un tableau, un formulaire et des blocs de contenu.

Footer

Le footer reprend la navigation principale et donne accès à la page “À propos”.

Responsive

La mise en page s’adapte aux écrans plus petits grâce aux media queries et au menu mobile.

5. Fonctionnalités prévues

Navigation entre les pages principales du site
Design responsive pour ordinateur, tablette et mobile
Tableau récapitulatif des compétences et débouchés
Formulaire de contact avec validation JavaScript
FAQ interactive sur la page contact
Filtres JavaScript sur la page vie étudiante
